# Yol Planı

## 1. Katalog Deneyimi
- [ ] Kitap sekmesindeki form ve tabloyu daha modüler, kart tabanlı bileşenlerle yeniden düzenle.
- [ ] Arama/filtre başlığını üst kısma taşıyıp hızlı istatistikler ekle.

## 2. Otomatik Kitap Bilgisi
- [ ] ISBN ve başlık aramalarından gelen verileri yayın adı, sayfa sayısı, basım yılı gibi alanlara otomatik yay.
- [ ] OpenLibrary'den dönen alt başlık/açıklama bilgisini not alanına önermek.

## 3. Kayıt Yönetimi
- [ ] Masaüstü uygulamasına güvenli kitap silme akışı ekle (aktif ödünç kontrolü + geri bildirim).
- [ ] Kategori seçimi ve geçmiş tablolarını yeni tasarıma uyumlu hale getir.

## 4. Yayın Hazırlığı
- [ ] Yeni tema ile tüm görsel bileşenleri doğrula, hata mesajlarını gözden geçir.
- [ ] Test senaryolarını (komut satırı testi + kullanıcı turları) güncelle.
